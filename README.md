
# Mc donalds order bot

Author: **Bohdan Ozarko**

## 📝 Description

Mc donalds order chat bot

## Prerequisites

1. Python 3.12+
2. Docker
3. Poetry 
4. Openai token

## 🖥 Usage

### How to run the application

1. ```cd mcdonalds_order_bot```
2. Create ```.env``` file and pass your openai key
3. ```./run.sh``` to run server
